# Camera Not Showing? Here's How to Fix It

## Quick Fixes

### 1. Click the START Button
- Look for the **START** button in the video player
- Click it to activate the camera
- Allow camera access when prompted

### 2. Check Browser Permissions
- **Chrome/Edge**: Click the lock icon in the address bar → Camera → Allow
- **Firefox**: Click the shield icon → Permissions → Camera → Allow
- **Safari**: Safari → Settings → Websites → Camera → Allow

### 3. Make Sure No Other App is Using Camera
- Close other apps that might be using the camera (Zoom, Teams, etc.)
- Restart your browser

### 4. For Mobile Devices
- **Streamlit Cloud**: Works automatically (HTTPS provided)
- **Localhost**: Camera access requires HTTPS on mobile browsers
- **Solution**: Use Streamlit Cloud (free) for mobile access

## Common Issues

### Black Screen
- This is normal during calibration (first 8 seconds)
- The screen will show your face after calibration
- If it stays black, check camera permissions

### "Camera not available" Error
1. Check if camera is working in other apps
2. Restart browser
3. Check system camera permissions (macOS: System Settings → Privacy → Camera)

### Camera Works But No Face Detection
- Make sure you're in good lighting
- Face the camera directly
- Remove anything covering your face
