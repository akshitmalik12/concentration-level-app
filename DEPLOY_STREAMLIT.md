# Deploy to Streamlit Cloud (Free & Simple)

## Step-by-Step Guide

### 1. Push to GitHub
Your code is already on GitHub at: `akshitmalik12/concentration-level-app`

### 2. Go to Streamlit Cloud
1. Visit [https://share.streamlit.io/](https://share.streamlit.io/)
2. Sign in with your GitHub account
3. Click "New app"

### 3. Configure Deployment
- **Repository**: Select `akshitmalik12/concentration-level-app`
- **Branch**: `main`
- **Main file path**: `app.py`
- **Python version**: 3.10 (or latest)

### 4. Deploy
Click "Deploy" and wait 2-3 minutes for the app to build and deploy.

### 5. Access on Mobile
- You'll get a URL like: `https://your-app-name.streamlit.app`
- Open this URL on your phone
- Allow camera access when prompted
